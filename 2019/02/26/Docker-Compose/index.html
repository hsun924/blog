Docker Compose

发表于 2019-02-26 | 更新于 2019-03-16 | 分类于 Docker

Docker Compose是官方编排项目之一，负责快速的部署分布式应用。允许用户通过一个单独的docker-compose.yml模板文件来定义一组相关联的应用容器为一个项目

Compose中有两个重要概念：

  • 服务(service) 一个应用的容器
  • 项目(project) 由一组关联的应用容器组成的一个完整业务单元
